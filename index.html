Project Overview

This project documents the design and implementation of a Romi robot capable of autonomously navigating and completing an obstacle course. The goal of the project was to integrate mechanical design, electrical components, and software control to create a reliable system capable of performing tasks without manual intervention.

The Romi robot was programmed using Python and incorporates several onboard sensors and control mechanisms to navigate the course. Key tools used by the robot include wheel encoders for tracking movement and distance, a line sensor for detecting and following lines on the course, and bump sensors for obstacle detection and collision response.

This portfolio provides a detailed overview of the project, including the mechanical and electrical design of the robot, the software architecture used to control it, and the analysis of its performance. Supporting materials such as code documentation, diagrams, photos, and a demonstration video are also included so that the system could be reproduced or further developed by others.
