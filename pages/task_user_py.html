task_user.py

Purpose

This file implements the USB serial user interface task.

Main Role

task_user.py gives the user live access to gains, speed settings, calibration routines, sensor printouts, and mode control.

Key Responsibilities

  • Read serial input
  • Dispatch commands to helper methods
  • Support gain entry and speed updates
  • Trigger calibration routines
  • Display sensor and estimator information
  • Control line-follow and course modes
