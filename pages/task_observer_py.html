task_observer.py

Purpose

This file implements the discrete-time observer used to estimate robot state variables.

Main Role

task_observer.py combines motor voltage inputs, wheel displacements, heading, and yaw rate to estimate forward position, heading, and wheel speeds.

Key Responsibilities

  • Read measured motor voltages, wheel displacement, heading, and yaw rate
  • Load discrete-time observer matrices from observer_matrices.py
  • Update the internal state estimate each sample
  • Publish estimated forward position, heading, and wheel speeds
  • Optionally stream estimated states through UART
  • Support internal path integration for debugging output
