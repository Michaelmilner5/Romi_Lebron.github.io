encoder.py

Purpose

This file defines the encoder class for quadrature position and velocity measurement.

Main Role

It provides wheel motion feedback for both motor control and state estimation.

Key Responsibilities

  • Configure timer channels in encoder mode
  • Track cumulative position
  • Handle counter wraparound
  • Estimate velocity from position change and time step
  • Support zeroing

Important Attributes / Shares / Variables

def get_velocity(self):
    return self.delta / self.dt if self.dt > 0 else 0.0

Why This File Matters

This file matters because encoder feedback is essential for repeatable wheel control and accurate observer inputs.
