task_line.py

Purpose

This file implements the line-following controller using centroid-based reflectance sensing and steering correction.

Main Role

task_line.py turns line-sensor measurements into left and right wheel commands so the robot can follow the printed path smoothly.

Key Responsibilities

  • Read normalized reflectance data from the line sensor array
  • Estimate line position using a centroid-style method
  • Apply filtering and confidence checks to the line-position estimate
  • Compute steering correction using PID-style logic
  • Clamp the steering command to prevent unrealistic wheel commands
  • Write wheel-command Shares used by the motor tasks
