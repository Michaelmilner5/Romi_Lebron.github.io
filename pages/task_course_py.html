task_course.py

Purpose

This file implements the high-level finite state machine for the obstacle course.

Main Role

task_course.py coordinates the full autonomous behavior of the robot. It decides when the robot should line-follow, drive straight, turn, back up, or stop based on course progress and sensor events.

Key Responsibilities

  • Define named states for the full course sequence
  • Enable and disable line-follow mode at the right times
  • Command direct wheel velocities for straight motion, turns, and backup maneuvers
  • Change gains and base speed depending on the phase of the course
  • Use measured wheel travel to trigger state transitions
  • Monitor the course-enable and bump-event Shares and respond safely
