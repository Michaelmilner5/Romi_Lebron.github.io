task_share.py

Purpose

This file defines the Share and Queue infrastructure used by the multitasking system.

Main Role

It provides the safe communication layer used by independent tasks.

Key Responsibilities

  • Define the BaseShare parent class
  • Implement Queue objects for buffered communication
  • Implement Share objects for latest-value communication
  • Maintain the global share list for diagnostics
