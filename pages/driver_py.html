driver.py

Purpose

This file defines the low-level BNO055 driver class.

Main Role

It handles raw I2C communication and exposes useful IMU operations to the higher-level IMU task.

Key Responsibilities

  • Read and write BNO055 registers
  • Verify the chip ID during initialization
  • Set operating modes
  • Read heading and gyro data
  • Read and write calibration coefficients
