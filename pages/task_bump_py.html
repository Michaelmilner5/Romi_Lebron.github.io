task_bump.py

Purpose

This file monitors the bump switches and turns a new press into a bump event.

Main Role

task_bump.py acts as the collision detector for the robot.

Key Responsibilities

  • Read left and right switch inputs
  • Detect new presses using edge logic
  • Set the bump-event Share when contact occurs

Important Attributes / Shares / Variables

NameDescription
_left / _rightInput pins for left and right bump switches
_bumpEventShare indicating a collision event
_prevL / _prevRPrevious switch states used for edge detection
