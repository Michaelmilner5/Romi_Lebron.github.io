task_imu.py

Purpose

This file reads and conditions heading and yaw-rate data from the BNO055 IMU.

Main Role

task_imu.py provides clean orientation information to the rest of the robot, especially the observer and debugging tools.

Key Responsibilities

  • Read heading and yaw rate from the IMU
  • Maintain a continuous heading through wraparound using angle unwrapping
  • Low-pass filter yaw-rate measurements
  • Reject unrealistic yaw-rate spikes
  • Support loading and saving IMU calibration data
  • Zero or rezero heading when needed

Why This File Matters

This file matters because raw IMU data is often noisy or wrapped in inconvenient ways. This task turns that data into a usable signal for estimation and debugging.
