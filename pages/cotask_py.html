cotask.py

Purpose

This file implements the cooperative scheduler and Task class.

Main Role

It is the timing backbone of the entire robot software system.

Key Responsibilities

  • Wrap generator-based tasks in Task objects
  • Schedule tasks by period and readiness
  • Support priorities, profiling, and tracing
  • Maintain the task list used in round-robin execution

Important Attributes / Shares / Variables

NameDescription
TaskClass representing one scheduled task
TaskListContainer for all tasks
priorityScheduling priority
periodExecution period in microseconds
go_flagFlag used to force a task ready
