task_garbage.py

Purpose

This file periodically runs garbage collection in the background.

Main Role

It helps manage MicroPython memory usage during runtime.

Key Responsibilities

  • Call the garbage collector inside a generator task
  • Run at a low priority so it does not interfere with core control loops
