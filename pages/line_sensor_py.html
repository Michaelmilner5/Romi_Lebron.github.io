line_sensor.py

Purpose

This file defines the LineSensorArray class used for the reflectance sensor array.

Main Role

It converts raw ADC values into usable reflectance information for line following.

Key Responsibilities

  • Create ADC objects for all sensor channels
  • Control the emitter pin
  • Read raw reflectance values
  • Support ambient cancellation
  • Store and apply white/black calibration
  • Compute normalized readings
