motor_driver.py

Purpose

This file defines the motor_driver class used to command the Romi motors.

Main Role

It abstracts direction, PWM, and enable control so higher-level tasks can request effort without managing pins directly.

Key Responsibilities

  • Configure PWM, direction, and sleep pins
  • Set signed motor effort
  • Enable the driver
  • Disable the driver

Important Attributes / Shares / Variables

NameDescription
nSLP_pinSleep/enable pin for the motor driver
DIR_pinDirection pin
