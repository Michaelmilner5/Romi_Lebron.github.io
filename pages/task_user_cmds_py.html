task_user_cmds.py

Purpose

This file contains the helper command logic used by the serial user interface.

Main Role

It keeps the command parser and helper routines separate from the main user task so the code stays organized.

Key Responsibilities

  • Print the help menu
  • Parse numbers for gains and speed
  • Stop the robot
  • Trigger line-sensor calibration
  • Display IMU and observer data
