task_start_button.py

Purpose

This file handles the physical start button used to start or stop the course.

Main Role

task_start_button.py provides debounced user control over the course-enable flag.

Key Responsibilities

  • Read the physical button input
  • Apply debounce timing
  • Toggle course-enable state
  • Clear bump event when starting a new run

Important Attributes / Shares / Variables

_btnInput pin for the start button
