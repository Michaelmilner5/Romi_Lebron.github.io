line_calib.json

Purpose

This file stores the saved white and black calibration values for the line sensor array.

Main Role

It allows the robot to load reflectance calibration values at startup instead of recalibrating every time.

Key Responsibilities

  • Store white-reference sensor values
  • Store black-reference sensor values
  • Support startup normalization in main.py and line_sensor.py

Important Attributes / Shares / Variables

NameDescription
whiteArray of white-surface calibration values
blackArray of black-surface calibration values
